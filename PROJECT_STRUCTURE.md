# بنية المشروع المحسّنة

```
src/
├── app/
│   ├── pages/
│   │   ├── Home.tsx              # الصفحة الرئيسية (يستخدم المكونات المقسّمة)
│   │   └── DesignSystem.tsx       # دليل نظام التصميم
│   ├── components/
│   │   ├── layout/
│   │   │   ├── Navigation.tsx     # شريط التنقل والقائمة الهاتفية
│   │   │   ├── MobileMenu.tsx     # القائمة المنسدلة للأجهزة الذكية
│   │   │   └── Footer.tsx         # التذييل
│   │   ├── home/
│   │   │   ├── HeroSection.tsx    # قسم البطل/الرئيسي
│   │   │   ├── StatsSection.tsx   # قسم الإحصائيات
│   │   │   ├── FeaturesSection.tsx # قسم المميزات
│   │   │   ├── ProgramsSection.tsx # قسم البرامج التدريبية
│   │   │   ├── PricingSection.tsx  # قسم الأسعار
│   │   │   └── CTASection.tsx      # قسم الدعوة للإجراء
│   │   ├── ui/                    # مكونات واجهة المستخدم المستوردة من shadcn
│   │   └── figma/
│   │       └── ImageWithFallback.tsx # مكون الصور مع بديل احتياطي
│   ├── App.tsx                    # المكون الرئيسي
│   └── routes.tsx                 # توجيه المسارات
├── constants/
│   ├── navigation.ts              # روابط التنقل وبيانات التذييل
│   ├── images.ts                  # عناوين URL الصور
│   └── home-data.ts               # بيانات الصفحة الرئيسية (إحصائيات، مميزات، برامج، أسعار)
├── hooks/
│   ├── useScroll.ts               # خطاف لكشف التمرير
│   └── useMobileMenu.ts           # خطاف لإدارة حالة القائمة الهاتفية
├── types/
│   └── index.ts                   # أنواع TypeScript المخصصة
├── utils/
│   └── icons.tsx                  # دوال مساعدة للأيقونات
├── styles/
│   └── index.css                  # الأنماط العامة
└── main.tsx                       # نقطة الدخول الرئيسية
```

## 🎯 المميزات الرئيسية

### 1. **فصل الاهتمامات (Separation of Concerns)**
- كل قسم من قسم الصفحة الرئيسية في مكون منفصل
- سهولة الصيانة والتحديث

### 2. **إدارة البيانات (Data Management)**
- جميع الثوابت والبيانات في مجلد `constants/`
- تسهيل تحديث المحتوى بدون تعديل المكونات

### 3. **Hooks مخصصة (Custom Hooks)**
- `useScroll` لكشف التمرير
- `useMobileMenu` لإدارة القائمة الهاتفية

### 4. **Type Safety**
- ملف `types/index.ts` يحتوي على جميع أنواع TypeScript
- سهولة الصيانة والكشف عن الأخطاء

### 5. **Utility Functions**
- مجلد `utils/` يحتوي على دوال مساعدة
- `icons.tsx` لإدارة الأيقونات

## 🚀 الفوائد

✅ **سهولة الصيانة** - كل مكون لديه مسؤولية واحدة  
✅ **إعادة الاستخدام** - المكونات يمكن استخدامها بسهولة  
✅ **قابلية التوسع** - إضافة صفحات وأقسام جديدة بسهولة  
✅ **أداء أفضل** - تقسيم الكود يحسن الأداء  
✅ **اختبار أسهل** - المكونات الصغيرة سهلة الاختبار  

## 📝 النقاط المهمة

- جميع المكونات تستخدم **React Functional Components**
- استخدام **Tailwind CSS** للأنماط
- استخدام **Motion (Framer Motion)** للحركات
- استخدام **React Router** للتوجيه
- دعم **RTL** (النصوص من اليمين إلى اليسار)
