# 💪 Professional Gym Landing Page

A modern, responsive web application for a professional fitness gym. Designed and developed with a focus on user experience and performance.

## 🎨 Features

- **Dynamic Landing Page** - Engaging sections with smooth animations
- **Responsive Design** - Fully optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX** - Dark theme with professional styling
- **Pricing System** - Flexible membership options
- **Testimonials** - Customer trust and reviews section
- **Smooth Navigation** - Interactive header with scroll effects
- **Mobile Optimized** - Smart action bar for mobile devices
- **High Performance** - Fast loading with Vite

## 🛠️ Tech Stack

- **React 18.3.1** - UI library
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **Radix UI** - Accessible UI components
- **Motion** - Smooth animations
- **React Router** - Client-side routing
- **React Hook Form** - Form management

## 📦 Installation

```bash
# Install dependencies
npm install
# or
pnpm install
```

## 🚀 Development

```bash
# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

## 🏗️ Building for Production

```bash
# Build the project
npm run build
```

## 📱 Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## ✨ Key Sections

- **Hero Section** - Eye-catching introduction
- **Stats Section** - Key metrics and achievements
- **Trust Section** - Member testimonials
- **Features Section** - Gym amenities and services
- **Programs Section** - Available fitness programs
- **Pricing Section** - Flexible membership plans
- **CTA Section** - Call-to-action for sign-ups
